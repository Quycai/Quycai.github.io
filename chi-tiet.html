XÁC NHẬN TẢI VỀ

- Cài đặt cơ bản: Tải file về -> Nhấn chuột phải chọn "Extract Here" (Giải nén) -> Vào thư mục vừa giải nén tìm file có đuôi .exe nháy đúp để chơi.

- Cách Fix lỗi chữ ô vuông: Bạn truy cập vào thư mục chứa game, tìm mục game/fonts. Hãy xóa file font mặc định bị lỗi đi, sau đó copy một font tiếng việt bất kỳ trên máy bạn (ví dụ: arial.ttf) dán vào thư mục đó, rồi đổi tên lại cho giống hệt tên file font gốc vừa xóa.
