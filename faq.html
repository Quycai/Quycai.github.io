💬 FAQ

Giải đáp thắc mắc và khắc phục các lỗi kinh điển khi cài game

🔑 Mật khẩu giải nén của các file game là gì?

Tất cả các file nén game (.rar hoặc .zip) được tải từ HPGVH18 Studio đều sử dụng chung một mật khẩu giải nén mặc định và duy nhất là: hpgvh18 (viết thường hoàn toàn, không dấu, không cách).

👾 Lỗi thiếu file hệ thống DLL (MSVCP140.dll, VCRUNTIME140.dll...) xử lý thế nào?

Đây là lỗi hệ điều hành Windows của bạn đang thiếu các gói thư viện vận hành môi trường game của Microsoft. Cách xử lý triệt để: Hãy lên Google tìm bộ cài mang tên "Visual C++ Redistributable All-in-One", tải về, giải nén và chạy file script cài đặt tự động để cập nhật toàn bộ hệ thống. Khởi động lại máy tính là có thể mở được game mượt mà.

🔲 Tại sao khi mở game lên chữ lại biến thành ô vuông (Lỗi font đậu hũ)?

Lỗi này do bộ engine gốc của game nước ngoài không nhận diện được hệ ký tự Telex Tiếng Việt. Để khắc phục, ông vào thư mục chứa game đã giải nén, truy cập đường dẫn game/fonts (nếu có). Xóa file font mặc định bị lỗi đi, sau đó copy một file font Việt hóa chuẩn trên máy tính (như Arial hoặc Roboto) vào và đổi tên nó trùng khít với tên file cũ vừa xóa là xong.

📱 Game dành cho nền tảng PC có chơi được trên điện thoại Android không?

Hiện tại các bản Patch ngôn ngữ của Studio được đóng gói tối ưu cấu trúc tốt nhất cho hệ điều hành Windows. Với các game phát triển bằng Ren'Py, ông có thể dùng ứng dụng giả lập JoiPlay trên Android để quét và chơi, tuy nhiên một số tính năng nâng cao hoặc khả năng hiển thị font chữ có thể bị hạn chế, không ổn định bằng chơi trực tiếp trên PC.

🔗 Link tải game bị sập, báo quá giới hạn lượt tải (Limit Drive) phải làm sao?

Nếu link Google Drive báo quá giới hạn tải trong 24 giờ, ông chỉ cần đăng nhập tài khoản Google của mình, bấm vào icon "Thêm lối tắt vào Drive" hoặc "Tạo bản sao" về tài khoản cá nhân của ông rồi tải xuống từ đó. Trường hợp link chết hoàn toàn (lỗi 404), hãy kéo xuống đáy trang chủ bấm nút báo lỗi hoặc nhắn qua Fanpage để tôi fix link luôn nhé.
