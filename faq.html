💬 FAQ

Giải đáp thắc mắc và khắc phục lỗi khi chơi game

🔑 Mật khẩu giải nén của các file game là gì?

Tất cả các file nén game (.rar hoặc .zip) được tải từ HPGVH18 Studio đều sử dụng chung một mật khẩu giải nén mặc định là: hpgvh18 (viết thường, không dấu).

👾 Lỗi thiếu file DLL (MSVCP140.dll, VCRUNTIME140.dll...) là sao?

Đây là lỗi cực kỳ phổ biến khi máy tính của bạn thiếu các thư viện chạy game cơ bản của Microsoft. Cách xử lý rất dễ: Hãy lên Google tìm từ khóa "Visual C++ Redistributable All-in-One", tải bộ cài về và chạy file để cập nhật toàn bộ hệ thống. Sau đó khởi động lại máy là chơi được mọi game.

🔲 Tại sao vào game chữ lại bị biến thành ô vuông (Lỗi đậu hũ)?

Lỗi này xảy ra do game gốc không hỗ trợ font chữ Tiếng Việt. Bạn hãy vào thư mục game đã giải nén, tìm đến thư mục game/fonts. Xóa font mặc định của game đi, tìm một font chữ chuẩn Tiếng Việt trên máy tính của bạn (như Arial, Roboto), copy vào đó rồi đổi tên file cho giống hệt tên font cũ vừa xóa. Hoặc có thể lên YouTube gõ tìm kênh PhongDHP123 để xem clip hướng dẫn từng bước nha.

📱 Game trên PC có chơi được trên Android không?

Hiện tại, các bản Patch Việt hóa của Studio chủ yếu tối ưu hóa cho hệ điều hành Windows (PC). Đối với các game chạy bằng công cụ Ren'Py, một số game có thể giả lập bằng ứng dụng JoiPlay trên Android, tuy nhiên độ ổn định và hiển thị font chữ sẽ không được hoàn hảo như trên máy tính.

🔗 Khi link tải game bị báo quá giới hạn hoặc lỗi 404?

Đối với link Google Drive, nếu bị báo lỗi quá giới hạn tải trong ngày, bạn hãy đăng nhập tài khoản Google của mình rồi bấm "Tạo bản sao" vào Drive cá nhân để tải về. Nếu link bị sập hoàn toàn (lỗi 404), hãy bấm ngay vào nút "Báo Cáo Lỗi Link" dưới chân trang hoặc nhắn tin qua Fanpage để Studio fix lại link sớm nhất.
